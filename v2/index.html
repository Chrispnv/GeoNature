A propos

Application de synthèse des observations faune et flore.

Elle regroupe toutes les données des différents protocoles FAUNE et FLORE en les limitant au niveau QUI QUOI QUAND OU.

Principe général : Un protocole = Un outil = Une BDD + Une BDD et une application de SYNTHESE regroupant les données des différents protocoles sur la base des champs communs à tous les protocoles (QUI a vu QUOI, OU et QUAND).

Code source, suivi des versions et des tickets sur https://github.com/PnEcrins/FF-synthese

Documentation (serveur, applications, base de données) sur http://ff-synthese.rtfd.org

Saisir

L'application permet de saisir les données contact faune et flore depuis une interface web.

Embarquer

Une application mobile permet de réaliser la saisie directement sur le terrain, embarquant les cartes IGN.

Centraliser

L'ensemble des données des différents protocoels sont stockées dans une base de données robutes et accessible en temps réel.

Partager

Les données centralisées peuvent être directement accessibles par les partenaires avec une gestion des droits.
